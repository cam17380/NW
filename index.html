Mode Navigation

CommandDescription

exitGo back one mode level

Configuration

CommandDescription
hostname <name>Set device hostname
ip address <ip> <mask>Set IP address on interface
no shutdownEnable interface (bring up)
shutdownDisable interface (bring down)
